# 마케팅 고객 데이터 분석 / 시각화

✅ **광고성과지표**

- **CTR(Click Through Rate)=클릭수 / 노출수 * 100**
- **CPM(Cost Per Mile)=광고비용 / 노출수 * 1000**
- **CPC(Cost Per Click)=광고비용 / 클릭수**
- **CPA(Cost Per Action)=광고비용 / 구매수**
